# Build/Run Instructions

# 1 - Prerequisites

* Java 11+
* An IDE (such as IntelliJ IDEA, Eclipse, or Visual Studio Code)

# 2 - Run the Program

* The main program can be found in KnapsnakDemo.java
* Compile and run from the project root:
  ```
  javac src/knapsnapProblem/*.java
  java -cp src knapsnapProblem.KnapsnakDemo
  ```

# 3 - Run Tests

* The test file can be found in knapsnapProblemTest.java
  ```
  java -cp src knapsnapProblem.knapsnapProblemTest
  ```
